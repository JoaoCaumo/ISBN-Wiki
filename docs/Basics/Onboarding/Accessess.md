* **github.com/SAP users** — Org membership on GitHub for SAP projects; enables repo access and SSO-controlled permissions.
* **VMGMT Scanning CON** — Vulnerability Management (VMGMT) group for running scans in the **CON** environment/segment.
* **VMGMT Scoping CON** — VMGMT group used to define/approve scan scope and assets in **CON**.
* **VMGMT Scanning ARI** — VMGMT scanner access for the **ARI** environment/segment.
* **VMGMT Scoping ARI** — VMGMT scoping/asset selection in **ARI**.
* **VMGMT Scanning FGL** — VMGMT scanner access for **FGL**.
* **VMGMT Scoping FGL** — VMGMT scoping/asset selection in **FGL**.
* **VMGMT Reporting ISBN-Comb…** — VMGMT reporting workspace (ISBN combined view); grants access to consolidated dashboards/exports.
* **ASR Tenable User** — Standard Tenable user role via ASR; view findings, dashboards, and assigned remediations.
* **PlusOne Netbox RW** — NetBox **read/write** access in PlusOne; allows editing IPAM/DCIM records.
* **Netbox Fieldglass Guest** — NetBox **guest/read-only** for Fieldglass-related inventory lookups.
* **GCID HSE CAE GCP Users** — Google Cloud Project (GCP) user group under GCID/HSE/CAE; baseline access to project resources.
* **Multicloud_Orca_User** — Orca Security user role for multicloud posture visibility and alert triage.

> **Note:** CON / ARI / FGL are listed as distinct environments or domains; if your org uses different names, update the labels accordingly.
